# KMPShip - Kotlin Multiplatform Starter Kit for Android & iOS

**KMPShip** is a production-ready Kotlin Multiplatform (KMP) boilerplate and starter template for building native Android and iOS apps with Compose Multiplatform. Skip weeks of setup and start shipping features from day one.

**Website:** [kmpship.app](https://www.kmpship.app/)
**Documentation:** [docs.kmpship.app](https://docs.kmpship.app/)

---

## What is KMPShip?

KMPShip is a fully configured Kotlin Multiplatform project template that gives you a solid foundation to build cross-platform mobile apps. Instead of spending time wiring up architecture, dependency injection, networking, authentication, and platform-specific integrations, KMPShip provides all of that out of the box so you can focus on your app's unique features.

## Key Features

- **Compose Multiplatform UI** - Share UI code across Android and iOS with Jetpack Compose
- **Clean Architecture** - Pre-configured layered architecture with clear separation of concerns
- **Dependency Injection** - Ready-to-use DI setup with Koin
- **Networking** - Ktor HTTP client configured for API communication
- **Firebase Integration** - Authentication, Firestore, push notifications, and more
- **In-App Purchases** - Payments and subscriptions for both platforms
- **In-App Reviews** - Prompt users to rate your app
- **Push Notifications** - Firebase Cloud Messaging setup for Android and iOS
- **App Store Publishing Guides** - Step-by-step instructions for Google Play and App Store releases
- **Light & Dark Mode** - Theming support built in
- **And more** - Image fetching, custom UI components, welcome screens, app icon configuration

## Who is KMPShip for?

- Solo developers and indie hackers who want to ship a cross-platform app fast
- Teams adopting Kotlin Multiplatform who need a proven project structure
- Developers looking for a Kotlin Multiplatform template with real-world features already integrated
